1. 各キャラクターの視界範囲を定義する

    -> 返ってきた視界情報をもとにグレートホワイトで視界の範囲をわかるようにする
  - サーバー側から返すステップ情報内に、キャラが見えているかどうかの情報を追加する
    -> DONE

2. 視界外の敵キャラ用のはてなキャラ画像を用意する
  -> DONE

3. 地形で階層が上の敵キャラクターははてなで表示
  -> できてないっぽい？チェック必要

### 視界の要件
* キャラクターの周囲8マスが視界範囲
* 高さが1上がるごとに視界が一マス増える
* 建物の裏にいるキャラクターは見えない
  - 建物から何マス離れているかによる
  - 建物から2マス離れている かつ 視認者の高さが建物の高さより1以上高い場合は見える
  - 建物から3マス離れている かつ 視認者の高さが建物の高さより2以上高い場合は見える みたいな感じで調整
